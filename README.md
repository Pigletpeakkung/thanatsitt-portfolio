## 🛠️ **Installation - Beginning the Ritual**

### 📋 **Prerequisites**
```bash

npm >= 8.0.0
```

```bash
git clone https://github.com/PigletPeakKung/thanatsitt-portfolio.git
cd thanatsitt-portfolio
```

### 📦 **Install Dependencies**
```bash
npm install
```

### 🚀 **Start Development Server**

npm run dev

## 🔧 **Customization - Making It Your Own**

### 🎨 **Color Themes**
Modify `tailwind.config.mjs` to change the color palette:

```javascript
// Add your own symbolic colors
colors: {
  'your-theme': {
    50: '#your-light-color',
    500: '#your-main-color',
    900: '#your-dark-color'
  }
}
```

### ✨ **Animations**
Create new symbolic animations in the Tailwind config:

```javascript
animation: {
  'your-ritual': 'your-ritual 3s ease-in-out infinite'
},
keyframes: {
  'your-ritual': {
    '0%, 100%': { /* start/end state */ },
    '50%': { /* middle state */ }
  }
}
```

### 🔮 **Components**
Add new Jungian-inspired components:

```astro
